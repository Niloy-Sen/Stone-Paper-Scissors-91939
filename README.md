# Stone-Paper-Scissors :tada: 
Stone Paper Scissors Game using JavaScript and HTML, CSS

### Rules

If the player wins, they gain 5 points. If the computer wins, the player loses 1 point.

- Paper beats Rock
- Rock beats Scissors
- Scissors beats Paper
